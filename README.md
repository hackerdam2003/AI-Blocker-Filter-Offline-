# AI Blocker Filter (Offline)

AI-based image protection tool that adds subtle adversarial noise to prevent AI models from accurately processing images.

## 🚀 Features
- Adds adversarial noise to images
- Protects images from AI detection
- Adjustable protection level
- Works completely offline
- Fast and lightweight

## 🛠️ How to Use

1. Open the tool in your browser
2. Click on "Choose file" and upload your image
3. Adjust the protection level slider
4. Process the image
5. Download or use the protected image
